# Persistent Task Manager

A command-line task manager built with Python.

## Features

* View tasks
* Add new tasks
* Mark tasks as completed
* Save tasks in a JSON file
* Reload saved tasks after restarting
* Handle invalid task numbers
* Handle missing or invalid JSON files

## Run

```bash
python app.py
```
